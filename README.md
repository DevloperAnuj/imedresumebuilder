# IMED Resume Builder

This is a hobby project designed to help students of **IMED (Institute of Management and Entrepreneurship Development)** build their institutional resumes hassle-free. It provides a clean, standardized format that aligns with institutional requirements, ensuring consistency and professionalism.

## Features
- **Easy Editing**: Inline editing for all sections (Personal Details, Education, Projects, Skills, etc.).
- **Real-time Preview**: Toggle between Edit and Preview modes to see exactly how your resume will look.
- **PDF Export**: One-click download as a perfectly formatted PDF.
- **Custom Header**: Includes the institutional header image by default.
- **Standardized Layout**: Margins, fonts, and spacing are pre-configured to meet guidelines.

## Tech Stack
- **React**: For the dynamic user interface.
- **Vite**: For fast development and building.
- **Tailwind CSS**: For styling and layout.
- **Lucide React**: For icons.

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/imed-resume-builder.git
    cd imed-resume-builder
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```
